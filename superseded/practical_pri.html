Practical Tasks - Primary Level

Note: This page contains sample program tracing tasks designed for primary school students. Students will be asked to trace through simple programs and predict their output. Actual tasks will be added closer to the contest date.

Program Tracing Overview

What is Program Tracing?

Program tracing involves following the execution of a program step by step to understand what it does and predict its output. This helps develop logical thinking and understanding of how computers execute instructions.

Skills Tested:

  • Following sequential instructions
  • Understanding loops and repetition
  • Tracking variable changes
  • Predicting program output

Sample Task Types

1. Simple Sequential Programs

Students will trace through programs with basic operations:

Example format:
x = 5
y = 3
result = x + y
print(result)

Question: What will be printed?

2. Programs with Loops

Students will trace through simple loops:

Example format:
for i in range(3):
    print(i)

Question: What will be the complete output?

3. Programs with Conditions

Students will trace through programs with if-else statements:

Example format:
age = 10
if age >= 12:
    print("Teenager")
else:
    print("Child")

Question: What will be printed?

4. Simple AI-Related Programs

Students will trace through programs that demonstrate basic AI concepts:

Example format:
scores = [85, 92, 78, 96]
total = 0
for score in scores:
    total = total + score
average = total / 4
print(average)

Question: What average will be calculated and printed?

Sample Practice Problems

Problem 1: Basic Operations (Easy)

Instructions: Trace through the following program and write down what will be printed.

robot_energy = 100
robot_energy = robot_energy - 20
robot_energy = robot_energy + 10
print("Robot energy:", robot_energy)

if robot_energy > 50:
    print("Robot is ready!")
else:
    print("Robot needs charging")

Your Answer: ___________________________

Solution: Robot energy: 90
Robot is ready!

AI Connection: This problem demonstrates how AI systems track and manage resources. Real AI robots use similar logic to monitor their battery levels and make decisions about when to recharge or continue working. The conditional logic (if-else) is fundamental to how AI makes decisions based on current conditions.

Problem 2: Loops and Counting (Medium)

Instructions: Trace through this program that counts smart devices in a home. Write down all the output.

smart_devices = ["phone", "tablet", "speaker", "watch"]
device_count = 0

for device in smart_devices:
    device_count = device_count + 1
    print("Found device:", device)

print("Total smart devices:", device_count)

if device_count >= 3:
    print("This is a smart home!")

Your Answer:

Line 1: ___________________________

Line 2: ___________________________

Line 3: ___________________________

Line 4: ___________________________

Line 5: ___________________________

Line 6: ___________________________

Solution:
Found device: phone
Found device: tablet
Found device: speaker
Found device: watch
Total smart devices: 4
This is a smart home!

AI Connection: This program shows how AI systems process and count data collections. Smart home AI assistants use similar loops to inventory connected devices, analyze usage patterns, and make recommendations. The counting and classification logic here is the same principle used in AI systems that manage Internet of Things (IoT) devices.

Problem 3: AI Decision Making (Hard)

Instructions: This program simulates an AI assistant making recommendations. Trace through it carefully and write down all output.

user_age = 12
user_interests = ["games", "science", "art"]
recommendations = []

for interest in user_interests:
    if interest == "games":
        if user_age < 13:
            recommendations.append("Educational games")
        else:
            recommendations.append("Strategy games")
    elif interest == "science":
        recommendations.append("Science experiments")
    elif interest == "art":
        recommendations.append("Digital art tools")

print("AI Recommendations for you:")
for recommendation in recommendations:
    print("-", recommendation)

total_recommendations = len(recommendations)
print("Total suggestions:", total_recommendations)

Your Answer:

Line 1: ___________________________

Line 2: ___________________________

Line 3: ___________________________

Line 4: ___________________________

Line 5: ___________________________

Solution:
AI Recommendations for you:
- Educational games
- Science experiments
- Digital art tools
Total suggestions: 3

AI Connection: This program demonstrates a simplified recommendation system, which is at the heart of modern AI applications like Netflix, YouTube, and Amazon. The nested conditional logic shows how AI considers multiple factors (age, interests) to personalize suggestions. Real recommendation AI uses much more complex algorithms, but the basic principle of analyzing user data to make tailored suggestions is the same.

Contest Format

During the contest, students will:

  • Receive printed program code similar to the examples above
  • Trace through the execution step by step
  • Write down the expected output
  • Answer questions about program behavior

Time Allocation: Approximately 30 minutes for program tracing tasks within the 1-hour contest.

Difficulty Levels:

  • Easy: Simple variable operations and basic conditionals
  • Medium: Loops with counting and list processing
  • Hard: Nested conditionals with AI-themed scenarios
